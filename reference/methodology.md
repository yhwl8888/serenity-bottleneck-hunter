## 2. 核心框架:供应链瓶颈逆向映射(THE CORE)

这是整套方法最值钱的部分,五步:

**第 1 步 · 锁定有确定性资本开支的大主题**
选一个"钱一定会砸进来"的结构性趋势(如 AI 算力扩建)。逻辑:需求的确定性 > 个股的故事性。

**第 2 步 · 逆向往上游拆产业链**
跳过人人都盯的下游龙头(如 $NVDA / $AVGO 的 ASIC),沿产业链一层层往上游追问"它依赖谁":
- 下游成品 → 中游组件 → **上游材料/设备/代工**。
- 越往上游,越可能存在"卡脖子"且"无人覆盖"的环节。

**第 3 步 · 三道埋伏闸门(必须同时满足)**

| 闸门 | 含义 | 验证问法 |
|---|---|---|
| 🔒 **真瓶颈** | 产能受限、有定价权、短期无法绕过 | "别人能在 1-2 年内 engineer around 它吗?" |
| 👁️ **前机构** | 分析师/算法还没覆盖,定价错位 | "卖方有几篇研报?散户听说过吗?" |
| 💰 **便宜 + 已去风险** | 估值压抑 + 下行有保护 | "PB/PS/远期 PE 多少?产能/订单是否已锁定?" |

**第 4 步 · 绑定催化剂**
找一个能让市场"突然看见"该标的的事件(行业大会、大厂签约、财报上修、ETF/纳入),在催化剂前埋伏。

**第 5 步 · 兑现与轮动**
催化剂兑现、机构涌入、估值重定价后,止盈或轮动到产业链上下一个尚未被发现的环节。

> **可直接作为 skill 的提问骨架**:主题 → 逆向拆链 → 定位瓶颈 → 过三道闸门 → 找催化剂。

## 3. 选股筛选清单(他反复使用的硬指标)

尽管追成长,他几乎每条论点都带估值锚:
- **估值**:P/E、P/S、P/B、远期 P/E、EV/FCF;偏好"低估的高增长"(如 Soitec 1×PB/2×PS"极度压抑";TSEM 2028 远期 PE ~17×)。
- **增长质量**:同比营收增速(常要 40%+ 乃至三位数)、毛利率(偏好 70%+)、远期营收订单/ARR、经营利润率。
- **供需结构**:产能是否被预订锁定(如 TSEM 70% 产能锁到 2028 = 下行保护)、是否有定价权/垄断。
- **筹码与情绪**:空头占比(高 SI = 轧空潜力,如 HIMS 42%)、流通盘大小(小流通盘易拉升也易被操纵)、解禁/稀释时间表。
- **客户质量**:客户是不是 Mag7 / 超大规模厂商(被巨头依赖 = 远期增长确定性)。

## 4. 催化剂与择时

### ★ 两套择时模式(经价格回测校准,2026-05)
对他 11 只光子学标的做了"首call日 → 之后峰值"的价格回测,发现**他有两套完全不同的择时**,切勿混用:

| 模式 | 适用 | 入场触发 | 回测证据 |
|---|---|---|---|
| **A. 主题瓶颈长线** | 结构性主题里的上游瓶颈股(光子学这类) | **主题刚点燃、早期上行/突破前高时就进,主动放弃抄底** | 11 只首call时多处于前 6 月区间的 **86%–237% 高位**(SIVE 在 237%,即已拉升中才进);之后 2-6 个月冲出 **+150%~+1100%**(中位 ~+277%) |
| **B. 波段/超跌反弹** | 成熟大票的错杀回调(2025 的 HIMS/SG/SMCI) | **买恐慌性、非实质性回调** | 见下方"回调分性质" |

> **关键纪律**:主题挖股(skill 主用法)用**模式 A**——他的 alpha 来自"早于机构发现主题",而不是抄到最低点。**等回调反而会踏空**(AXTI 首call后直接 +828%,从不回头)。"买回调"只适用于模式 B 的成熟票。

### 其余择时要点
- **催化剂驱动,且严格区分真假**:
  - ✅ 真:标普纳入资金流、ETF 上线、财报远期营收上修、大厂签约、行业大会(如 NVDA GTC 推光子学)。
  - ❌ 假(常是给空头的噪音):CFO 离职、措辞强硬的监管信、无实质的会议。
- **忽略情绪,提前埋伏**:散户在消息落地时恐慌,他在之前布局("政府停摆/降息早已 price in")。
- **回调分性质**:
  - 非实质性下跌(如广告用语监管信)= 买点;
  - 实质性下跌(增发、稀释、远期营收受损)= 回避。
- **二阶 / 三阶效应**:对宏观事件不买大路货,而是顺着供应链推到最受益的冷门标的(见 §8 委内瑞拉案例)。
- **极端情绪反向**:"最好的买点是散户割肉的极度恐慌时";新叙事狂热时反而减仓。

## 5. 仓位、风控与期权

- **集中 + 高信念**:单一标的可占组合 ~30%(如 NBIS),用 DCA 分批建仓而非一次满仓。
- **期权工具箱**:
  - **LEAPS / 看涨期权**:博不对称上行(IV 低时性价比高);
  - **备兑看涨(CC)/ 现金担保看跌(CSP)**:在已持有的优质资产上做被动现金流(卖远价外、避开财报周);
  - **对冲**:$VIRT / VIX 作为不对称波动率对冲。
- **按贝塔定杠杆**:高贝塔(NBIS/RKLB)约 1.3–1.5× 保证金;低贝塔(META/AMZN)可更高。
- **只在愿意持有的标的上卖看跌**:卖 put = 等于愿意在该价位接货,绝不在投机仙股上卖权。
- **税务优化意识**:长期资本利得、年底税务亏损收割 + "一月效应"、(超高净值)C-corp 交易结构。

## 6. 思维模型与行为原则

1. **信息发现 > 跟随情绪**:读一手资料(财报、电话会、申报文件),"忽略情绪,因为它通常是错的"。
2. **少用技术分析**:画线仅用于"同板块公司的模式押韵"(history rhymes),而非预测。
3. **被巨头依赖 = 护城河**:寻找"Mag7 不得不用它"的小公司(ALAB、NBIS 的逻辑)。
4. **资本开支会向下游漏斗**:钱从巨头流向被忽视的上游供应商,这是反复使用的母题。
5. **用回报率%而非金额衡量**:刻意不晒美元金额/豪车,以"论点 → 数月后股价验证"自证,反"卖课网红"。
6. **透明实时喊单**:提前给出确切仓位,而非事后诸葛。

## 7. 回避清单(以及为什么)

- **脱离基本面的"信仰股"**:$PLTR、$TSLA(估值与基本面脱钩)。
- **零/微营收却高市值的题材股**:量子($RGTI $QBTS $IONQ)、部分核能($OKLO)——"几乎没营收却 100-200 亿+ 市值"。
- **净值代理 / 持币壳公司**:$BMNR、$MSTR 类——"想要敞口不如直接买底层资产 ETH/BTC"。
- **被高估的大盘**:在无回调时追高的拥挤交易(如他对满仓高 beta 看涨期权的警告)。
- **有实质抛压的标的**:正在大额增发/解禁的公司。

- 关键去风险信号:**高于现价的融资**(如 NBIS 在现价之上拿到融资 = 看涨信号)。

### 8.3 地缘事件的供应链二阶效应(委内瑞拉案例)
- 不买大路货原油 ETF,而是顺着重质油产业链推:化肥/氨($CF)、能炼重质油的炼厂($VLO $PSX)、海军监视/无人机($LDOS $AVAV $LHX $HII)、不良主权债重组顾问($ASHM $HLI $LAZ)、稀释剂物流($TRGP)。
- 复用要点:**任何宏观冲击,都问"谁是那个被忽视的、最直接受益的上游/配套环节"**。

## 9. 风险与局限(必须诚实写明)

- **业绩自报、未经审计**:+1,116% / +3,840% YTD、两年 +22,561% 等均为社媒自述,大概率来自微盘 + 期权 + 杠杆的高方差打法,**不可线性外推**。
- **幸存者偏差**:他展示的"我喊过的 10x 清单"是结果筛选,未兑现的标的不在其中。
- **微盘 + 冷门 = 高流动性风险 + 高稀释风险**(他本人也反复提防稀释)。
- **风格不可照搬**:集中 + 期权 + 杠杆,普通投资者复制极易爆仓。
- **方法的有效性依赖"信息差"**:一旦某环节被广泛覆盖,alpha 即消失,必须持续往新瓶颈轮动。

## 10. 把它变成可执行工作流(skill 的程序雏形)

给定一个**主题或标的**,让 Claude 按以下步骤产出分析:

1. **确认主题的资本开支确定性**:这股钱为什么一定会花?规模多大?周期多长?
2. **逆向拆解供应链**:列出从下游到上游的每一层环节及代表公司。
3. **定位瓶颈层**:哪一层产能受限/有定价权/短期无法绕过?
4. **过三道闸门**:对候选标的逐一检验【真瓶颈 / 前机构 / 便宜且已去风险】。
5. **匹配催化剂**:未来 3-12 个月有什么事件能让市场"看见"它?
6. **风险清单**:稀释、解禁、流动性、竞争替代、估值透支。
7. **输出**:候选标的 + 一句话瓶颈逻辑 + 关键估值锚 + 催化剂 + 风险 + 强制免责声明。
